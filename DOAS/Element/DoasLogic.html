Dehumidification Control

Entry Conditions

  • OAT > 55°F (minimum threshold)
  • Outdoor RH > 85% (high threshold)
  • Indoor/Lab RH > 55% (high threshold)
  • Space temp within 3°F of setpoint

Exit Conditions (with hysteresis)

  • OAT ≤ 55°F
  • Outdoor RH < 83% (low threshold)
  • Indoor/Lab RH < 53% (low threshold)
  • Cooling system failure (both stages)

Dehumidification Operation Sequence

  • Phase 1: Test Stage 1 cooling effectiveness (1 min)
  • Phase 2: If Stage 1 effective (5°F drop), add heating
  • Failover: If Stage 1 fails, test Stage 2 (1 min)
  • Exit: If both stages fail, exit dehumidification mode
  • Safety: Heating only added after cooling proven effective

Control Function Overview

doasControl(metricsInput, settingsInput, currentTempArgument, stateStorageInput)

The main control function processes incoming sensor data and generates control commands for the DOAS unit with proper mode switching, occupancy control, and safety interlocks.

Input Parameters

  • metricsInput: Sensor data (temps, humidity, etc.)
  • settingsInput: Equipment/location ID, setpoints
  • currentTempArgument: Current supply temperature
  • stateStorageInput: State persistence for timers

Control Outputs

  • Fan enable/disable
  • Heating enable/disable
  • Gas valve position (0-100%)
  • Cooling enable/disable
  • DX Stage 1/2 enable
  • Outdoor damper position
  • System status and alarms
